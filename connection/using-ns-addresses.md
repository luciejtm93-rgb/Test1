# 🛰 Используя NS-адреса

> Быстрый ответ: откройте домен, нажмите `Замена NS-записей`, введите новые NS через пробел и дождитесь обновления.

## Как сменить NS

1. Откройте `Мои домены`
2. Выберите нужный домен
3. Нажмите `Замена NS-записей`
4. Введите новые NS в формате:

```text
ns1.example.com ns2.example.com
```

5. Подтвердите изменение

## Важные правила

- бот поддерживает до 20 NS
- все NS должны быть рабочими
- новые NS не должны совпадать с текущими
- вводите NS через пробел, не через запятую

## Сколько ждать

Обычно NS меняются до 1 часа. Редко — до 24 часов.

## Если после смены NS сайт не работает

Это не всегда ошибка. После смены NS ещё может открываться старая страница или сайт может не открываться только у части пользователей.
