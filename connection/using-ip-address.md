# Используя IP-адрес

> Быстрый ответ: если у вас есть IP сервера, откройте домен и используйте `Добавить A-запись` или создайте запись в `1REG DNS`.

## Когда этот способ подходит

Используйте IP-адрес, если:

* у вас есть сервер или VPS
* хостинг дал вам IPv4-адрес
* вы хотите быстро направить домен на сервер

## Как сделать в боте

### Вариант 1: при покупке

Во время покупки выберите `Добавить A-запись` и введите IP сервера.

### Вариант 2: после покупки

1. Откройте домен
2. Нажмите `1REG DNS`
3. Нажмите `Создать запись`
4. Выберите `A`
5. Укажите IP сервера

## Если нужен Cloudflare

Если хотите использовать Cloudflare, откройте [Подключение Cloudflare в 1 клик](../cloudflare/connect-cloudflare.md).

## Важно

`1REG DNS` публикует реальный IP сервера. Если для вас важнее защита и скрытие origin IP, используйте Cloudflare.
