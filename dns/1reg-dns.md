# Что такое 1REG DNS

> Быстрый ответ: `1REG DNS` — это встроенное управление DNS-записями прямо в боте. Оно подходит для быстрого создания записей, но публикует реальный IP сервера.

## Что важно знать сразу

`1REG DNS` публикует реальный IP. Если ваш IP уже в блок-листах или вы не хотите светить origin IP, лучше использовать Cloudflare.

## Что можно делать в 1REG DNS

* просматривать текущие записи
* создавать записи
* удалять запись нажатием на неё

## Какие типы записей доступны

Сейчас в `1REG DNS` доступны:

* `A`
* `AAAA`
* `CNAME`
* `TXT`

## Как быстро применяются изменения

Обычно изменения применяются примерно через 5–10 секунд после изменения.

## Когда лучше использовать Cloudflare

Используйте Cloudflare, если вам нужны:

* скрытие origin IP
* кэш
* Auto SSL
* базовая защита

Откройте [Подключение Cloudflare в 1 клик](../cloudflare/connect-cloudflare.md).
