# Управление API: токен, whitelist и webhook

> Быстрый ответ: API не заработает, пока whitelist пустой. Сначала добавьте разрешённый IP, потом используйте токен.

## Что видно в API-управлении

* токен
* статус API-доступа
* лимит запросов
* статус webhook
* список whitelist IP

## Как включить API

1. Откройте `Управление API`
2. Добавьте IP в `Whitelist`
3. Убедитесь, что доступ больше не заблокирован
4. Используйте токен в своей интеграции

## Важные детали

* если whitelist пустой, доступ к API запрещён
* текущий лимит — `10 RPS`
* webhook настраивается отдельно
* при пересоздании токена старый токен нужно заменить в интеграциях

## Что делать, если API не работает

Проверьте:

* добавлен ли IP в whitelist
* не используете ли старый токен
* не упираетесь ли в лимит запросов
